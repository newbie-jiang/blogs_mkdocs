安装库

```
sudo apt-get install lzop
```

解压

```
tar -vxjf linux-imx-rel_imx_4.1.15_2.1.1_ga_alientek_v2.2.tar.bz2
```

- 正点原子写了一个编译脚本，直接运行即可

```
./mx6ull_alientek_emmc.sh
```
