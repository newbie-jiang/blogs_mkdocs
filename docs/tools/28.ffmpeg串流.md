安装ffmpeg

```c
# 开放 9999/UDP（如用 UFW）
sudo ufw allow 9999/udp

# 用 ffmpeg 监听并送出 SRT（原样不转码）
ffmpeg -re -stream_loop -1 -i 800x480_30.mp4 \
  -c copy -f mpegts \
  "srt://0.0.0.0:9999?mode=listener&latency=120000&pkt_size=1316"
```

**说明**

- `latency=120000`（120ms）可按网络状况调大些（如 200–300ms）提升抗抖动。
- SRT 走 UDP
